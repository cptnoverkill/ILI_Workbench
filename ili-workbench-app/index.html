🔁 Re-parent CF IDs to Smallest Containing Cell

Checks each correlated feature's measured position (distFt / clock) against all CF cells. If a smaller nested cell also contains that point, the correlation is re-parented to it and the corrId is reassigned using the next available suffix for that cell.

⚙ Set Pipe OD for Rendering

Sets the pipe OD used to render all feature boxes on the heatmap. Imported ILI and Creaform width values are not changed. Correlation result geometry (width, length, depth) is recalculated from each feature's grid selection under the new OD.

New OD (in)

Sessions saved before V0.2.3 may have incorrect Dist (ft/in) and/or Clock values. This tool recalculates all geometry fields from the saved grid bounds for every correlation.
